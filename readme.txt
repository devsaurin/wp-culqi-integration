Culqi es una pasarela de pago para Peru. Este plugin permite una integración con todos los cargos usando al API de Culqi.

== Description ==

Culqi Integration te permite sincronizar todos los pagos con tu Wordpress.

Con este plugin podrás:

* Visualizar todos los pagos como un CPT, y ver el detalle de cada uno.
* Si usas Woocommerce podrás activar el método de pago para este fin.
* Activar un log de actividades de Culqi para cada pedido de Woocommerce.


== Installation ==
1. Unzip and Upload the directory 'wp-culqi-integration' to the '/wp-content/plugins/' directory

2. Activate the plugin through the 'Plugins' screen in WordPress

3. Put your Public Key and Secret Key
